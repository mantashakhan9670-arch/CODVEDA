# Task 1: Simple Calculator

## Description

Develop a basic calculator that performs the four primary arithmetic operations: **addition, subtraction, multiplication, and division**.

## Objectives

* Create separate functions for each arithmetic operation:

  * Addition
  * Subtraction
  * Multiplication
  * Division
* Take two numbers as input from the user.
* Allow the user to choose the desired arithmetic operation.
* Display the calculated result.
* Handle division by zero by displaying an appropriate error message instead of causing the program to crash.

## Expected Output

The calculator should:

1. Accept two numbers from the user.
2. Display a menu of available operations.
3. Perform the selected operation.
4. Show the result or an error message if the user attempts to divide by zero.
